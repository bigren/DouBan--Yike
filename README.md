# DouBan--Yike
豆瓣一刻APP单页面应用
 
前端框架使用了AngularJs将项目的数据与试图分离,并通过AJAX实时获取服务器后台接口数据.(注:应豆瓣一刻APP以与2017-8-16停止运营所以最多只能获取到当日的数据)
